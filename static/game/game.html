Laser Game for Laptops and PCs

Available for free!

Looking for something fun? This dodge game is inspired by the popular game Subway Surfers. You play as a brick trying to dodge blue squares and destroy bricks with lasers. Control the brick using the keyboard keys A or D, or the left/right arrows. To shoot, press the spacebar or arrow up.

The game features a highscore board and more. It may not have top-notch graphics, but it’s perfect for small children. Best of all, it’s free and doesn’t require WiFi. Updates are released occasionally, so check back for new versions. Currently, the game is unavailable for smartphones and macOS—we apologize for the inconvenience.
